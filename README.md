# Project Name
E-commerce mvc spring project
## Description
E-commerce web application built with Spring MVC and Hibernate.

## Technologies Used
- Java
- Spring MVC
- Hibernate
- SQL Server
- JSP
- HTML / CSS

## Features
- User Registration & Login
- product magement (add,create,detele,edit)
- Browse Products
- Add to Cart
- Checkout
- Order Management
- Payment (Fake)

## Database Tables
- Users
- Products
- Cart / Cart Items
- Orders / Order Items
- Payment

## How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Configure database in `hibernate.cfg.xml`
4. Run the project on Tomcat Server
